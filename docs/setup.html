Setup Guide

Get SportDeck running on your local machine or shared hosting in a few steps.

Prerequisites

Before installing SportDeck, make sure you have the following:

🐘
PHP 7.4+

With PDO and PDO_MySQL extensions enabled.

🗄️
MySQL 5.7+ / MariaDB

Any modern MySQL-compatible database server.

🖥️
Web Server

XAMPP, MAMP, Laragon, or Apache/Nginx.

Installation

1

Copy the project folder

Place the sportdeck folder inside your web server's root directory:

/xampp/htdocs/sportdeck/        ← XAMPP (Windows)
/Applications/MAMP/htdocs/sportdeck/  ← MAMP (macOS)
/var/www/html/sportdeck/        ← Apache (Linux)
2

Start your web server

Open XAMPP / MAMP and start both Apache and MySQL services.

Database Setup

3

Import the database schema

Option A — phpMyAdmin (recommended):

  1. Open http://localhost/phpmyadmin
  2. Click Import in the top menu
  3. Select sportdeck/db.sql and click Go

Option B — MySQL CLI:

mysql -u root -p < /path/to/sportdeck/db.sql
ℹ️ db.sql automatically creates the sportdeck_db database and inserts sample data including admin and player accounts.

Configuration

4

Edit includes/config.php

Open the config file and update your database credentials and base URL:

// Database connection
$db_host = 'localhost';
$db_user = 'root';       // your MySQL username
$db_pass = '';           // your MySQL password
$db_name = 'sportdeck_db';

// Base URL — change if installed in a different subfolder
$base_url = '/sportdeck';
⚠️ If you rename the project folder (e.g. to sports), update $base_url to match: '/sports'
5

Open in your browser

http://localhost/sportdeck/

You should see the SportDeck homepage listing sample tournaments.

⚠️ Change all default passwords before deploying to any public server.

Admin Panel

After logging in as admin, navigate to /sportdeck/admin/dashboard.php. From here you can:

  • 📋 Dashboard — Overview of tournaments, matches, users
  • 🏆 Tournaments — Create, edit, and update tournament status
  • 👥 Teams — Add teams with coach name, linked to a tournament
  • 🏃 Players — Add players to team rosters
  • 📅 Matches — Schedule matches, set venue & time
  • 📊 Results — Record scores and winners
  • 👤 Users — View all registered accounts

Player Portal

Logged-in players are redirected to /sportdeck/player/dashboard.php. Players can:

  • 🔍 Browse all available tournaments on the homepage
  • ✅ Register for open tournaments (with player cap support)
  • ❌ Cancel their own registrations
  • 📋 View their registrations at /player/my_registrations.php
  • 📅 View fixtures and match results publicly

Tips & Notes

  • 💡 The public pages (homepage, fixtures, results) are accessible without login.
  • 💡 auth.php guards all admin and player pages — no direct URL access without login.
  • 💡 The migrate.php file can be used to re-run schema migrations if needed.
  • 💡 All SQL queries use PDO prepared statements to prevent SQL injection.
  • 💡 Passwords are stored using PHP's password_hash() with PASSWORD_DEFAULT.
